# Brute Force Authentication Project

## Overview
This project demonstrates a brute force authentication attack in a controlled ethical hacking lab environment using Kali Linux and Hydra.

## Features
- Simple login authentication page
- Brute force simulation using Hydra
- Password security analysis
- Preventive controls demonstration

## Technologies Used
- Kali Linux
- Apache2
- PHP
- Hydra
- GitHub

## Preventive Controls
- Strong Passwords
- Multi-Factor Authentication
- CAPTCHA
- Rate Limiting
- Account Lockout

## Educational Purpose
This project is developed strictly for educational and ethical hacking learning purposes.
